# Fontes Gotham (self-host)

Para a Gotham aparecer na Vercel e para todos os visitantes, coloque aqui os arquivos **.woff2** da Gotham (desde que sua licença permita uso web).

Sugestão de nomes e pesos:

| Arquivo           | Peso (weight) | Uso        |
|-------------------|---------------|------------|
| Gotham-Book.woff2 | 400           | body/texto |
| Gotham-Medium.woff2 | 500         | (opcional) |
| Gotham-Bold.woff2 | 700           | bold       |
| Gotham-Black.woff2 | 900          | títulos    |

Se os nomes dos seus arquivos forem outros, avise para ajustarmos o `globals.css`.

Enquanto estes arquivos não forem adicionados, o site usa **Plus Jakarta Sans** (Google Fonts) como fallback, uma geométrica parecida com Gotham.
